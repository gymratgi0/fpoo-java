JAVA

While

O while é uma estrutura de repetição. Enquanto a condição for verdadeira, a estrutura continuará se repetindo. Ele só sai do laço, se a condição for falsa.

Switch

O switch é usado para o usuário escolher uma das opções propostas.

If/Else

É uma estrutura de condição, onde se a condição for verdadeira ele irá realizar uma ação, se for falsa ele irá realizar outra ação.
